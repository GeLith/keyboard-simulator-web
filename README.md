# 键盘输入模拟器 Chrome 扩展 v1.0.0

这是一个浏览器扩展，用于模拟键盘输入，适用于不支持粘贴输入的网页。

GitHub: https://github.com/GeLith/keyboard-simulator-web

## 安装方法

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `keyboard-simulator` 文件夹
5. 扩展安装完成

## 功能特点

- 页面右上角显示蓝色球形悬浮图标
- 点击球形图标展开操作面板
- 支持开始延迟设置（秒）
- 支持字符输入间隔设置（秒）
- 支持拖动球形图标位置

## 使用方法

1. 在任意网页，点击右上角的蓝色球形图标
2. 输入要模拟输入的文字
3. 设置延迟时间和字符间隔
4. 点击"开始输入"按钮
5. 在延迟时间内切换到目标输入框
